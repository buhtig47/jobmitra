How to delete your JobMitra data

JobMitra (package com.jobmitra.app, developer Ayush Srivastava) respects your right to remove the data we hold about you. This page explains exactly what we store and how to request deletion.

What data we store on our servers

  • A randomly generated install ID (anonymous UUID, no personal info)
  • Firebase Cloud Messaging (FCM) token, used only to send push notifications
  • Profile preferences you entered during onboarding: state, education level, category, age, preferred job types
  • Jobs you saved or marked as applied
  • Smart alert rules you created (keywords, state filters, etc.)

We do not store your name, date of birth, phone number, or any document numbers on our servers. Those fields, when filled in the "Personal Info" section of the app, live only on your phone (Android SharedPreferences) and are used solely for the local form-fill cheatsheet PDF.

How to request deletion

Email us from the address you want associated with the request. Include your install ID if you can find it (Profile → About → tap "App Version" five times to reveal it), or simply the device you used.

Send to: ayushsrivastava47@gmail.com
Subject: JobMitra Data Deletion Request

What we delete

  • All rows tied to your install ID and FCM token are permanently removed from our database
  • Saved jobs, alert rules, and profile preferences linked to that ID
  • FCM token unsubscribed so you receive no further push notifications

How long it takes

Requests are typically processed within 7 working days. You will receive a confirmation email once your data has been deleted.

What stays

  • Anonymous, aggregated analytics (Firebase Crashlytics crash reports, AdMob ad impressions) that cannot be tied back to you personally.
  • Server access logs (Cloud Run, Cloudflare) auto-expire after 30 days as part of standard infrastructure retention.
